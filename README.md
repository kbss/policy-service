# Getting Started

## Stack
 - java 17
 - spring boot
 - REST Assured
 - mongodb
 - swagger
 - docker

## Prerequisite
1. Install Docker

## How to
1. `docker-compose up`
    1. Will start mongodb server
    2. Build and start web application
2. Open http://localhost:8080/swagger-ui/index.html#/
## After
1. Shutdown
 - `docker-compose down`
